Esfriando uma taça de café¶

  • Consideremos uma caneca isolante cilindrica que possui um liquido (agua ou caf) a uma temperatura inicial $T_0$ que homogenea no cilindro.

  • Pensamos que a temperatura evolui com o tempo $t$ e que homogenea nas fatias horizontais. Logo, nossa incgnita a temperatura $T\equiv T(t,y)$ onde $y\in [0,h]$ representa a altrua desde o fundo da caneca.

Obs: em geral, se considera que a temperatura homogenea para todo tempo $t$ (ou seja, $T$ independente de $y$) e tem-se uma lei ODE efeitiva para a evoluo da temperatura com $t$: Lei de resfriamento de Newton $$ \frac{dT}{dt} \propto (T_{amb}-T)$$ onde $T_{amb}$ a temperatura ambiente (assumida constante).

  • vamos fazer um outro modelo sem assumir independencia de $y$ baseado na eq. de difuso do calor: $$ \frac{\partial T}{\partial t} = c \frac{\partial^2 T}{\partial y ^2}$$ com $c$ ("thermal diffusivity").
  • as condies de contorno/iniciais so: $$ T(t=0, y) = T_0 \ \forall y $$ $$ \partial_yT(t, y=0) =0, \forall t \text{ nao tem transferencia de calor no fundo}$$ $$ T(t,y=h) = T_{amb} $$

Obs: $c=1.43 \times 10^{7} m^2/s$ para agua a $25^oC$. O comportamento com a temperatura : Podemos assumir constante no nosso problema $T\in[25,80]$

Construimos a solução aproximada com diferencias finitas¶

usamos 2 pontos "ghost" em $y=-\Delta y$ e $y=h +\Delta y$ para incorporar as condies de bordo

def evol_temp_cafe(nt,nx,dt,dx,c,m,u,unew,uxx,Tamb):
    uts = []
    uts.append(u)
    for im in range(m):
        for it in range(nt):
            uxx[1:nx-1] = u[2:nx] - 2*u[1:nx-1] + u[:nx-2]
            uxx /= dx ** 2
            unew =  u + dt * c * uxx
            #condicoes de contorno
            unew[0]=unew[2] # d_x u(t,y=0) = 0, ix=0 ponto ghost
            unew[nx-1] = Tamb # u(t,y=h) =Tamb
            u = unew
        uts.append(u) # guardamos u cada Tmax transcurrido
    return uts

O modelo melhorou!!¶

Estrutura populacional em função do tempo¶

Mathematical Models in Biology, Christina Kuttler, Seo 4.2

  • possveis idades $a > 0$
  • numero de individuos com idade $a$ a tempo $t$: $u(a,t)$
  • mortalidade dos individuos de idade $a$ por unidade de tempo: $\mu(a)$. Em geral, assume $\mu(a)\geq \mu_0 > 0$.
  • EDP: $$ \partial_t u (a,t) + \partial_a u (a,t) = - \mu(a) u(a,t) $$ McKendrick-von Foerster equation

Obs: uma eq. de transporte (de "velhice")

  • condio inicial: $u(a,t=0) = u_0(a)$ (distribuio inicial conhecida)

  • condio de contorno: nascimentos $$ u(0,t) = B(t)$$

  • em geral, complicado modelar $B(t)$ pois precisa de conhecer a taxa de nacimento $\beta(a)$ na idade $a$ e a populao $u(a,t)$! ("condio de contorno no-local" em $a$)

  • um modelo simplificado ("renewal equation") $B(t)$ soluo de: $$ B(t) = \int_0^t \beta(a) B(t-a) e^{-\int_0^a \mu(s) ds}da+\int_t^\infty \beta(a)u_0(a-t)e^{-\mu(a)t}da$$

Obs: notar que esta uma eq. integral no trivial para $B(t)$!

  • para fazer a simulao numerica, discretizamos $B_j:= B(j\Delta t)$ com $$ B_0= B(0) = \int_0^\infty \beta(a)u_0(a) da $$ e aproximamos a integral por uma soma ($j\geq 1$) $$ B_j = \beta(0) B_j \Delta t + \sum_{k=1}^j \beta(k\Delta t) B_{j-k} e^{-\int_0^{k\Delta t}\mu(s)ds} \Delta t + \int_{j\Delta t}^\infty \beta(a)u_0(a-t)e^{-\mu(a)t}da $$
  • colocamos $\Delta a = \Delta t$ na discretizao

Cuidado:¶

  • a equao $\partial_t u + \partial_a u = f$ pode ser instvel numricamente dependendo dos valores de $\Delta t, \Delta a$
  • a aprox. numerica desenvolve singularidades em alguns casos
  • um criterio de estabilidade $0\leq \Delta t/\Delta a \leq 1$ ("Courant number") [ Ref]

Eq. de Poisson 2d¶

  • incgnita $u:D \subset \mathbb{R}^2 \to \mathbb{R}$ $$ \partial_x^2 u + \partial_y^2 u = f(x,y)$$ com $f$ funo dado.
  • veremos caso particular de condies de bordo (Dirichlet): todos os nodos do bordo esto dados $$ u|_{\partial D} = h : \partial D \to \mathbb{R}$$

Discretizaçao num dominio retangular $D=[0,L_x]\times [0,L_y]$¶

  • Eq. de Poisson discreta
  • consideramos discretizao com $\Delta x = \Delta y$, $$ u(i,j) := u(i \Delta x, j \Delta y),\ i=0,..,n_x-1, \ j=0,..,n_y-1$$
  • os valores $u(i,j)$ correspondientes ao interior de $D$ podem ser compilados em um vetor (ordenados "naturalmente") $$ U = (u(1,1),u(2,1),u(3,1),...,u(n_x-2,1),u(1,2),u(2,2),...) $$ de tamanho $(n_x-2) (n_y -2)$
  • pensando no laplaciano discreto, a eq. em diferenas finitas toma a forma: $$ \delta^2_x u + \delta^2_y u (i,j) = \frac{1}{\Delta x^2} ( u(i+1,j)+u(i-1,j)+u(i,j+1)+u(i,j-1)-4u(i,j)) = f(i,j)$$
  • as condies de bordo so $u(i,j) = h(i,j)$ para $(i,j)$ representando pontos no bordo
  • pode-se juntar todas as eqs. algebricas resultantes em um grande sistema linear da forma $$ A U = b $$ onde $A$ uma matriz quadrada com $(n_x-2) (n_y -2)$ linhas, que espara com entradas $0,-1$ ou $4$, e $b$ um vetor que tem of $f(i,j)$ correspondentes ao interior de $D$ assim como os $h(i,j)$ correspondentes ao bordo
  • como $A$ espara, o sistema pode-se resolver em muitos casos, obtendo a aproximao numrica $u(i,j)$ para $u(x,y)$

Obs: uma modelagem concreta consiste em que $u(x,y)$ descreve o potencial eletrico gerado por uma distribuio de carga $\rho(x,y)$ via $f(x,y) = -\rho(x,y)/\epsilon_0$ (wiki).

Se tem uma superficie conductora $\Sigma$, $u$ tem uma condiao de bordo nesta superficie: $u|_\Sigma$ constante (suponhamos zero se o condutor est conectado 'a terra').

interessante achar o potencial eletrico dado por diversas distribuicoes de carga e configuracoes de superficies condutoras (verificar 'cargas imagens').

Um modelo bi-dimensional para ondas sismicas (feito pelo Prof. Ricardo)¶

  • consideramos uma onda acstica propagandose na rocha, cuja amplitude na posio $(x,z)$ ($z$ profundidade) a tempo $t$ $p(x,z,t)$
  • a propagao desta onda est modelada pela EDP (tipo eq. de onda) $$ \ddot{p}(x,z,t) \ = \ c(x,z)^2 (\partial_x^2 p(x,z,t) + \partial_z^2 p(x,z,t)) \ + s(x,z,t) $$
  • a velocidade de propagao $c(x,z)$ depende de datos geologicos:

velocidade para tipos de rocha comuns:

Rock Type Velocity (m/s)
Unconsolidated Sandstone 4600 - 5200
Consolidated Sandstone 5800
Shale 1800 - 4900
Limestone 5800 - 6400
Dolomite 6400 - 7300
Anhydrite 6100
Granite 5800 - 6100
Gabbro 7200

Discretizando:¶

  • Indices acima denotam direo $t$ e embaixo denotam direoes $x,z$ $$ \frac{p_{j,k}^{n+1} - 2 p_{j,k}^n + p_{j,k}^{n-1}}{\mathrm{d}t^2} \ = \ c_j^2 ( \partial_x^2 p + \partial_z^2 p) \ + s_{j,k}^n $$

  • para achar os $p_{j,k}^{n+1}$, propomos

$$ p_{j,k}^{n+1} \ = \ c_j^2 \mathrm{d}t^2 \left[ \partial_x^2 p + \partial_z^2 p \right] + 2p_{j,k}^n - p_{j,k}^{n-1} + \mathrm{d}t^2 s_{j,k}^n $$

onde as derivadas espaciais so:

$$ \partial_x^2 p \ = \ \frac{p_{j+1,k}^{n} - 2 p_{j,k}^n + p_{j-1,k}^{n}}{\mathrm{d}x^2} $$$$ \partial_z^2 p \ = \ \frac{p_{j,k+1}^{n} - 2 p_{j,k}^n + p_{j,k-1}^{n}}{\mathrm{d}z^2} $$

c = np.zeros((nz, nx))

if model_type == "homogeneous":
    c += c0
elif model_type == "fault_zone":
    c += c0
    c[:, nx // 2 - 5: nx // 2 + 5] *= 0.8    
elif model_type == "surface_low_velocity_zone":
    c += c0
    c[1:nz//20,:] *= 0.8
elif model_type == "random":
    pert = 0.4
    r = pert * np.random.rand(nz, nx)
    c += c0 * (1 - r)
elif model_type == "topography":
    c += c0
    c[0:nz//20, nx//20 : 5*nx//20] = 0                         
    c[0:nz//20, 10*nx//20 : 12*nx//20] = 0                  
    c[0:3*nz//20, 14*nx//20 : 16**nx//20] = 0
    c[2*nz//20: 4*nz//20, 2*nx//20: 4*nx//20]  = 0
    c[0 : 2*nz//20, 2*nx//20 : 10*nx//20] *= 0.8    
elif model_type == "slab":
    c += c0
    c[11*nz//20 : 12*nz//20, 0 : 12*nx//20] = 1.4 * c0
    for i in range(11*nx//20, 15*nx//20):

# Time evolution
for it in range(nt):
    if nop==3:
        # calculate partial derivatives, be careful around the boundaries
        pxx[:, 1:nx-2] = p[:, 2:nx-1] - 2*p[:,1:nx-2] + p[:, :nx-3]
        pzz[1:nz-2, :] = p[:nz-3, :] - 2 * p[1:nz-2, :] + p[2:nz-1, :]      
    elif nop==5:
        # calculate partial derivatives, be careful around the boundaries
        pxx[:, 2:nx-2] = -1./12*p[:,4:nx]+4./3*p[:,3:nx-1]-5./2*p[:,2:nx-2]+4./3*p[:,1:nx-3]-1./12*p[:,:nx-4]
        pzz[2:nz-2,:] = -1./12*p[4:nz,:]+4./3*p[3:nz-1,:]-5./2*p[2:nz-2,:]+4./3*p[1:nz-3,:]-1./12*p[:nz-4,:]
            
    pxx /= dx ** 2
    pzz /= dz ** 2

    # Time extrapolation
    pnew = 2 * p - pold + dt ** 2 * c ** 2 * (pxx + pzz)
    # Add source term at isx, isz
    pnew[isz, isx] = pnew[isz, isx] + src[it]

    # Plot every isnap-th iteration
